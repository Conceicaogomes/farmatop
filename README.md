# 💊 FarmaTop

A FarmaTop é uma farmácia fictícia desenvolvida em React + Vite com foco em experiência moderna de e-commerce.

O projeto foi criado para praticar:
- Componentização
- React Hooks
- Responsividade
- Carrinho de compras
- Favoritos
- Dark Mode
- Deploy online

---

# 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS3
- React Icons
- Git
- GitHub
- Vercel

---

# ✨ Funcionalidades

✅ Carrinho de compras  
✅ Sistema de favoritos  
✅ Busca de produtos  
✅ Dark mode  
✅ Layout responsivo  
✅ Menu mobile  
✅ Toasts/notificações  
✅ Finalização via WhatsApp  
✅ Deploy online
